# kalandralang-mode

Major mode for editing [kalandralang](https://github.com/doomeer/kalandralang) files.

## Automatic Indentation

Indentation implemented with
[SMIE](https://www.gnu.org/software/emacs/manual/html_node/elisp/SMIE.html).

Once `kalandralang` supports better formatting, we will also add support for
auto-formatters such as [apheleia](https://github.com/radian-software/apheleia).
